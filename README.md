 Skill Exchange Platform

 Overview

Skill Exchange Platform is a web-based application that connects individuals who want to share, learn, and exchange skills. The system allows users to offer their expertise, request assistance in specific areas, and interact with other members to promote knowledge sharing and community collaboration.

The platform is designed to create a collaborative environment where users can teach skills they possess and learn new skills from others.

---

 Objectives

- Facilitate skill sharing among users.
- Promote peer-to-peer learning.
- Connect individuals with complementary skills.
- Provide an easy-to-use platform for exchanging knowledge and expertise.

---

- User Profile Management

- Post Skill Offers

 Installation Guide

 Step 1: Clone the Repository

  git clone https://github.com/YOUR_USERNAME/MyProject.git


 Step 2: Move to Project Directory

  cd MyProject


 Step 3: Place Project in XAMPP Directory

Move the project folder to:

C:\xampp\htdocs\


 Step 4: Start Apache and MySQL

Open XAMPP Control Panel and start:

- Apache
- MySQL

 Step 5: Create Database

  Open phpMyAdmin:
  
http://localhost/phpmyadmin

Create a database named:

  sql
  skill_exchange

 Step 6: Import Database

Import the SQL file provided in the project.

 Step 7: Run the Application

Open your browser and visit:

http://localhost/MyProject
